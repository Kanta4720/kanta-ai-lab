# news.json Specification

This document outlines the new specification for the `news.json` file, designed to support the AI News Brief site.

## Root Object Structure

The root object contains three keys:

| Key                | Type   | Description                                                                 |
| ------------------ | ------ | --------------------------------------------------------------------------- |
| `generated_at_jst` | String | The timestamp when the file was generated (JST).                            |
| `todays_brief`     | Array  | An array of the top 5 most important news articles of the day.              |
| `categories`       | Object | An object containing news articles grouped by category.                     |

## Article Object Structure

Each article object, whether in `todays_brief` or under a category, has the following structure:

| Key             | Type   | Description                                                                 |
| --------------- | ------ | --------------------------------------------------------------------------- |
| `title`         | String | The original headline of the news article.                                  |
| `summary_2lines`| String | A 2-line summary of the article, generated by AI.                           |
| `why_it_matters`| String | An AI-generated explanation of why the news is important (context/background). |
| `market_impact` | String | An AI-generated analysis of the impact on markets, companies, or the economy. |
| `category`      | String | The category of the news (e.g., "Markets", "Tech", "Geopolitics").        |
| `source`        | String | The original source of the news (e.g., "Reuters", "Bloomberg").             |
| `url`           | String | The URL to the original article.                                            |

## Example

```json
{
  "generated_at_jst": "2026-02-21 12:00",
  "todays_brief": [
    {
      "title": "Global Tech Stocks Rally on AI Chip Breakthrough",
      "summary_2lines": "A major semiconductor firm announced a new AI chip, promising a 10x performance leap. The news sent tech stocks soaring globally.",
      "why_it_matters": "This breakthrough could accelerate AI adoption across all industries, marking a significant milestone in computational power.",
      "market_impact": "Chip manufacturer stocks surged over 15%. The ripple effect is expected to boost the entire tech sector and related ETFs.",
      "category": "Tech",
      "source": "Bloomberg",
      "url": "https://www.bloomberg.com/"
    }
  ],
  "categories": {
    "Tech": [
      {
        "title": "Global Tech Stocks Rally on AI Chip Breakthrough",
        "summary_2lines": "A major semiconductor firm announced a new AI chip, promising a 10x performance leap. The news sent tech stocks soaring globally.",
        "why_it_matters": "This breakthrough could accelerate AI adoption across all industries, marking a significant milestone in computational power.",
        "market_impact": "Chip manufacturer stocks surged over 15%. The ripple effect is expected to boost the entire tech sector and related ETFs.",
        "category": "Tech",
        "source": "Bloomberg",
        "url": "https://www.bloomberg.com/"
      }
    ],
    "Markets": []
  }
}
```
